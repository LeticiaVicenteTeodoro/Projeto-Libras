É super fácil aprender a jogar! Siga as instruções:

  1. Clique no card do tema que deseja jogar.
  2. Ao abrir, vão aparecer dez cards relacionados a esse tema, cada um com seu respectivo nome.
  3. Faça a linguagem de sinais correspondente a cada nome conforme foi aprendido nas aulas anteriores.
  4. Após fazer, clique no card para ver o vídeo e conferir se acertou.

O objetivo do jogo é acertar o máximo possível.
